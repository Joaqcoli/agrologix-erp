Impactar product_units.stock_qty sumando cantidad comprada

Recalcular product_units.avg_cost usando promedio ponderado para ESA unidad:

new_avg = (old_stockold_avg + qtyunit_cost) / (old_stock + qty)

Guardar histórico como hoy (si existe).

Pedidos (aprobación / descuento)

Usar product_units.avg_cost como costo para margen.

Si stock queda negativo: permitir (como hoy), pero marcar alertas en Stock tab.

Intake (Carga Pedido)

Si el parser detecta producto + unidad:

Validar que exista product_units para esa unidad.

Si el producto existe pero esa unidad no existe → marcar “Unidad no válida” y pedir corrección en preview.

Si el producto no matchea, continúa como “No Product Found” como hoy.
